# 欢迎来到熊窝～

😃欢迎来到熊窝！这是一个部署在 GitHub 上，通过 Hexo 实现的个人博客！如果你持续关注这个网站，你将会收到关于熊的各种动态以及一些关于 PhotoShop、前端学习、java 的心得或是分享！😉这是这个博客成立的第一天，这也是熊窝里的第一篇文章！

# 为什么要建立它？

假期貌似延长了呀🤷‍♂️，而且自打我开始学前端，被 css 的功能震撼到以后，就非常想哪天搞一个人的网页 (没有精确到一个博客)，后来被我哥推荐了 Hexo 后直接打开新世界。有兴趣的小伙伴一定要试一试 Hexo，或许过几天我会总结一篇关于使用 Hexo 并将网站部署在 GitHub 上的文章🎉

当然你觉得我在扯淡的话就理解为闲的没事

# 这里将会出现什么样的文章？

如果你持续关注这个网站，你将会收到关于熊的各种动态以及一些关于 PhotoShop、前端学习、java 的心得或是分享！😉这是这个博客成立的第一天，这也是熊窝里的第一篇文章！

这是写在前面的话～等我玩明白了 hexo 打算开三个模块，分别关于上面所提到的 PhotoShop、前端的学习心得、大一时学的 Java 杂谈 (大概率会更新极慢或者直接弃坑) 以及我分享的吉他谱或是生活琐事

事实上这也是一篇测试文章，这个窝还有太多不成熟的地方，但我很期待以后的发展

# 广告时间！

😜欢迎各路天南海北喜欢 Minecraft 的小伙伴来到 "熊猫国度" 服务器！

这是一个由我和我室友共同组建的一个 Minecraft 轻度 RPG 生存服务器！👏基于 IA 插件，尽力挖掘 Minecraft 中存在的更多原创可能，并将它们带到玩家的眼前！

1831 年 10 月 28 日，法拉第在一次会议上展示了他发明的圆盘发电机，它是利用电磁感应的原理制成的，是人类历史上第一台发电机．据说，在法拉第表演他的发电机时，一位贵妇人问道 “法拉第先生，这东西有什么用呢？” 法拉第答道 “夫人，一个刚刚出生的婴儿有什么用呢？”

🐱‍👤这是一个由开服不入流的两个大学生运营的服务器，虽然可能会有各种各样的缺陷，😊但正因为我们都对这一领域感到陌生并乐于向往，因此我们的对于服务器的建设思路将偏向奇特，大家都认可并新奇的玩法才是最终的目标！
